# 🏥 TAUP Medical Center

TAUP Medical Center is a modern healthcare website that integrates Artificial Intelligence (AI) to provide faster, smarter, and more accessible medical services. The platform allows users to explore departments, communicate with an AI health assistant, view doctors, read medical news, and book appointments online.

## 🌐 Live Demo

You can view the deployed website here:

**https://taup-sszb.vercel.app**

## ✨ Features

- 🏠 Modern landing page
- ℹ️ About Us section
- 🤖 AI Health Assistant
- 🩺 Medical Departments
- 👨‍⚕️ Specialist Doctors
- 📍 Contact page with map
- 📰 Medical News
- 📅 Online Appointment Booking
- 📱 Responsive and user-friendly design

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Vercel (Deployment)
